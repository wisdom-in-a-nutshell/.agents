# Adi Writing Mode — Short Post

Use for:
- LinkedIn posts
- public notes
- short reflections
- concise website or personal-brand style writing

## Default shape
- start with the main point
- keep one core idea
- use line breaks for readability
- make it feel human, not performative

## What usually works
- direct statement
- one useful insight
- one concrete example or lived observation
- restrained conclusion

## Watchouts
- do not sound like a personal-brand machine
- do not force fake inspiration
- do not over-format with gimmicky spacing

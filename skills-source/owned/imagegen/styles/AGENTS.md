# AGENTS.md - Image styles

This folder holds reusable visual families for the `imagegen` skill.

## What belongs here

- reusable project style guides
- mascot / visual canon files that should carry across projects
- stable visual families worth reusing later

## What does not belong here

- per-image worklogs
- raw prompt experiments
- generated image outputs

## How to use it

When a new image task begins, check whether a relevant style guide already exists here before inventing a new visual direction.

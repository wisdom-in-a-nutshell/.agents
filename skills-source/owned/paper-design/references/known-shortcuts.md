# Known Paper shortcut notes

This is **not** a complete canonical shortcut sheet. It is only what could be verified from Paper's public docs/build log.

## Documented shortcuts and modifiers

- `Alt R` — Copy as React CSS
- `Alt T` — Copy as Tailwind
- `⌘` while resizing an image on macOS — crop image
- `Ctrl` while resizing an image on Windows — crop image
- `Alt`-double-click on resize handles — resize element to fill its parent
- double-click on resize handles — resize element to fit its content

## Notes

- Paper's public docs do not currently expose a full shortcut reference page.
- The build log also mentions fixes involving the `I` hotkey, numeric hotkeys for opacity and zoom, and Enter/Esc repeat behavior, but those changelog notes are not enough to safely reconstruct a full shortcut sheet.
- When answering shortcut questions, prefer these verified items first and say plainly when the official docs are incomplete.
